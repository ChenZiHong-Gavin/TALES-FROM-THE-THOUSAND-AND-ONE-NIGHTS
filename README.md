## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/ChenZiHong-Gavin/chinese-old-movie.git
   ```
2. Start the front-end and back-end projects separately

    Front-end:

    ```sh
    npm install
    npm start
    ```

     Back-end:
    ```sh
     mvn clean install
    ```
